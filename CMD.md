# CMD 常用指令

### I.　查閱過去輸入過的指令
```
history
```

### II.　列出當前目錄文件
```
dir /b
```

### III.　清除輸出
```
cls
```

### IV.　切換路徑
- 上個路徑　`cd..`
- 指定路徑　`cd xxx\xxx\xxx`
- 切換至該槽根目錄　`cd\`
- 切換磁碟(D)　`D:`

### V.　電腦名稱
```
hostname
```

### VI.　查詢本機 IP 位址
```
ipconfig
```

### VII.　[CMD 修改檔案之常見語法](https://blog.csdn.net/zmzhangcsdn/article/details/107298516)
- 插入 a / s
- 不儲存退出 : `:q`　/　強制不儲存退出 : `:q!`
- 儲存退出 : `:wq`　/　強制儲存退出 : `:wq!`
- 取消動作 : `esc`
