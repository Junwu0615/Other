# 檔案總管如何統一欄位格式

## STEP.1　自訂欄位
開啟任一檔案總管之資料夾，將它們排列成需要的欄位格式。<br/>
如 `示意圖 1` 內容所示，選定了 `名稱` / `大小` / `修改日期` / `類型` 。<br/>
